Melihat Halaman man di Web Browser

Jika ingin tahu penggunaan sebuah program pada Linux, secara tradisional rujukannya adalah halaman manual yang bisa dilihat menggunakan perintah man nama_program dari command line. Jika pada mode grafis, tentu saja membaca manual melalui perintah man dilakukan melalui terminal emulator, atau program seperti xman. Alternatif lain adalah membaca halaman manual melalui web browser, tapi bagaimana caranya? Berikut ini adalah cara yang saya tempuh.

Agar bisa membaca halaman manual melalui web browser, saya menggunakan dua program, yaitu man2web (http://man2web.sourceforge.net/) sebagai konverter yang memproses halaman man ke HTML, dan program mongoose (http://code.google.com/p/mongoose/) sebagai web server sederhana.

mongoose saya pilih sebagai web server untuk keperluan ini karena cukup kecil dan sederhana. Jika akan menggunakan web server lain yang sudah tersedia pada komputer, tentu juga bisa. Agar bisa membaca halaman manual melalui web browser, caranya tak sulit. Donwnload program man2web, lakukan kompilasi, kemudian letakkan dalam direktori yang mengizinkan menjalankan cgi script pada web server. Jalankan web server, lalu buka web browser dan isikan ke alamat man2web yang sudah disimpan pada web server, misalnya http://localhost/cgi/man2web.cgi

Untuk yang menggunakan mongoose, setelah mongoose terinstal, jalankan web server mongoose. Simpan program man2web dengan nama man2web.cgi pada direktori yang menjadi direktori web untuk program mongoose. Setelah itu tinggal mengetik alamat http://localhost:8080/man2web.cgi pada address bar web browser, maka halaman manual bisa disajikan pada web browser.

Untuk saya, membaca halaman manual melalui web browser (memakai Firefox) lebih enak. Hurufnya bisa ditampilkan lebih besar, mencari sebuah teks pada halaman manual yang sedang dibaca juga cukup mudah, dengan memanfaatkan pencarian yang ada pada web browser.
