Merias Text Editor

Judul di atas adalah kalimat campur aduk. Seharusnya "Merias Editor Teks", itupun kalau editor adalah kata yang benar dalam bahasa Indonesia. Sudahlah. Yang sebenarnya saya lakukan adalah melakukan sedikit pengaturan pada program jEdit yang menjadi program utama saya dalam menulis kode program. Karena menulis kode program adalah pekerjaan yang banyak saya lakukan, tentu saja alat bantu untuk tugas ini harus enak dan nyaman digunakan.

Pengaturan yang saya lakukan adalah pemilihan font dan mengatur warna untuk syntax higlighting. Font yang cocok untuk menulis kode program adalah font yang mudah dibaca, jelas, dan dalam kategori monospace. Perihal font monospace, ternyata ada juga orang yang tidak menyetujuinya. Ada yang mengatakan dengan font proporsional bisa juga menulis kode program dengan baik. Namun sampai saat ini saya tidak berminat mencoba untuk menggunakan font proporsional untuk menulis kode program.

Ada banyak font monospace yang bisa digunakan. Berikut ini adalah beberapa pilihan yang sempat saya coba. Inconsolata, Envy R, dan TI92pluspc adalah pilihan yang baik.

DejaVu Sans Mono
Bitstream Vera Sans Mono
Inconsolata
TI92pluspc
Envy R
Courier

Masih belum puas setelah menggunakan salah satu font yang dipilih, masih ada cara untuk menyempurnakannya. Menggunakan program fontforge, maka karakter (tepatnya glyph) yang kurang pas bisa diperbaiki penampilannya sesuai selera.

Selesai dengan urusan font, pengaturan kedua adalah pemakaian warna untuk syntax highlighting. Sebelumnya saya menggunakan latar belakang putih dengan syntax highlighting default yang ada pada jEdit. Untuk pemakai kaca mata tebal 8-), penampakan seperti ini ternyata kurang nyaman. Latar belakang gelap akhirnya memberikan rasa yang lebih enak.
