Template Presentasi dengan HTML

Membuat presentasi untuk ditampilkan di layar komputer (atau melalui proyektor), tidak harus menggunakan program khusus presentasi. Memang umumnya orang menggunakan program seperti OOo Presentation, atau mungkin yang paling dikenal seperti Power Point, atau bisa juga menggunakan LaTex.

Tetapi tidak ada salahnya membuat presentasi dengan program lain. Bisa saja dengan Wordprocessor dengan membuat huruf besar-besar. Atau dengan menampilkan gambar yang sudah disusun dengan model presentasi. Atau juga dengan cara lainnya. Salah satu kemungkinan adalah menggunakan HTML. Lalu dilihat dengan web browser, dalam mode fullscreen.

Untuk keperluan tersebut, tentu cukup memudahkan jika dibuat sebuah template HTML yang bertujuan untuk tampil selayaknya sebuah presentasi. Saya mencoba untuk membuat template tersebut. Ketika akan menyajikan presentasi sederhana, kita tinggal membuat isinya. Dengan memakai HTML, yang diperlukan untuk menampilkan presentasi ini hanyalah sebuah web browser. Jadi tidak bergantung pada sebuah program presentasi khusus.

Silahkan download template presentasi dengan ukuran sekitar 6 KB. Ekstrak, lalu buka file index.htm, dan ikuti contoh presentasi yang berisi cara menggunakan template tersebut.
