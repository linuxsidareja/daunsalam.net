Daftar Aplikasi untuk Sistem Operasi GNU/Linux

Baik untuk yang baru mencoba menggunakan Linux, atau yang sudah lama memakainya, tentu memerlukan berbagai aplikasi dalam menyelesaikan sebuah tugas. Pertanyaan seperti ini juga beberapa kali mampir pada saya. Dimana bisa mendapatkan daftar aplikasi yang berjalan di sistem operasi Linux.

Tempat yang sering saya kunjungi untuk keperluan ini adalah http://directory.fsf.org/ dan http://www.linux.org/apps/. Cukup banyak situs di internet yang menyajikan daftar seperti ini. Silahkan dilihat pada alamat berikut:

  • http://lsm.execpc.com/lsm/ Linux software map
  • http://www.boutell.com/lsm/ di sini menyediakan form pencarian untuk Linux software map
  • http://www.icewalkers.com/
  • http://www.ibiblio.org/pub/linux/
  • http://www.sai.msu.su/sal/ daftar aplikasi untuk Linux yang berkaitan dengan sains
  • http://linuxappfinder.com/
  • http://www.linuxrsp.ru/win-lin-soft/table-eng.html berisi tabel yang memuat daftar aplikasi di Windows dan persamaannya untuk Linux, atau sebaliknya

Walaupun saya bukan pengguna distro Debian, namun ada satu yang sangat membantu saya. Daftar aplikasi yang dimiliki distro ini luar biasa banyaknya. Nah daftar inilah salah satu favorit saya ketika memerlukan software apa yang cocok untuk sebuah keperluan. Untuk mengambil daftar ini bisa di alamat http://www.us.debian.org/distrib/packages Daftar yang berupa file teks ini disimpan di harddisk, kemudian untuk menemukan sebuah aplikasi tinggal gunakan grep beserta kata kunci yang sesuai. Andaikan ingin dimasukkan ke dalam database, seperti MySQL, tentu juga bisa. Pencarian data dilakukan dengan query melalui sebuah script sederhana, misal dengan PHP/Perl.
