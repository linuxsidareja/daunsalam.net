Menu Transparan pada Fluxbox

Jika anda menggunakan window manager fluxbox pada Linux, ada konfigurasi untuk membuat menu jadi transparan. Untuk mengatur tingkat transparan bisa melalui fluxbox menu -> configure -> transparency. Nilai yang disediakan untuk alpha berkisar dari 0 sampai 255. Nilai 0 berarti transparan 100%, sedangkan 255 tidak transparan sama sekali. Nilai 160 cukup baik untuk dicoba sebagai permulaan.

Setelah hal di atas dilakukan, reload konfigurasi fluxbox, maka menu akan menjadi transparan. Namun tidak selalu hal ini cukup untuk menghasilkan menu yang transparan. Ada beberapa hal yang perlu diperhatikan agar berhasil.

  • Versi fluxbox harus lebih besar dari 0.9.2 dan harus dikompilasi dengan opsi Xrender enable. Demikian juga dengan X server, ekstensi Xrender sudah enable. Untuk memeriksa kedua hal ini, jalankan perintah fluxbox -i pada konsol, dan lihat apakah ada RENDER pada hasil yang ditampilkan. Untuk informasi Xrender pada X, gunakan perintah xdpyinfo
  • Background harus diset dengan program yang sudah bisa memanfaatkan Xrender, lihat informasi ini dengan perintah fbsetbg -i
  • fbsetbg adalah program yang disertakan beserta fluxbox, yang berguna untuk setting background layar, seperti gambar untuk wallpaper. Ketika dijalankan, fbsetbg mencari program lain yang melakukan aksi sesungguhnya untuk menampilkan gambar sebagai wallpaper
  • Program yang bisa melakukan setting untuk wallpaper yang dipakai oleh fbsetbg salah satunya adalah feh
  • Untuk dapat menjalankan atau kompilasi feh memerlukan library imlib2 dan giblib

Setelah hal di atas ada, maka untuk menampilkan wallpaper dilakukan dengan perintah fbsetbg -f /path/ke/file/gambar Atur nilai alpha pada konfigurasi transparency fluxbox, dan load lagi konfigurasi fluxbox. Semoga menu yang tampil sekarang sudah transparan.
