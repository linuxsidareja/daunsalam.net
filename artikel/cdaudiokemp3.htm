Membuat File mp3 dari CD Audio

Tulisan ini dibuat sederhana dan ringkas. Artikel ini dibuat karena satu hari kemarin tiba-tiba mempunyai keinginan mejadikan beberapa lagu yang ada dalam CD audio menjadi file mp3. Kemudian disimpan di harddisk untuk dinikmati sewaktu bekerja di komputer.

Ada dua tahap yang dilakukan ketika menjadikan lagu yang ada dalam CD audio menjadi file mp3. Langkah pertama mengambil data lagu di CD audio untuk dijadikan dalam format wav. Langkah ini biasa juga disebut dengan "grab". Tahap ke dua menjadikan file wav ini menjadi file mp3. Proses ini disebut dengan "encode".

Banyak software yang tersedia untuk proses ini. Ada juga yang bisa langsung melakukan proses pengambilan data dari CD audio dan merubahnya ke file mp3. Namun secara detail, software ini tetap melakukan dua langkah di atas, yaitu "grab" dan "encode".

Di internet banyak tersedia software yang bisa melakukan konversi dari CD audio ke bentuk file mp3. Percobaan yang dilakukan di sini menggunakan software audiograbber versi 1.83 dan lame versi 3.96.1. Kedua software ini adalah software gratis. Audiograbber dapat diambil di http://www.audiograbber.com-us.net sedangkan lame di situs http://www.mp3dev.org

Langkah pertama yaitu melakukan konversi data lagu dari CD audio menjadi format wav. Masukkan CD audio ke cd drive, kemudian jalankan software audiograbber. Dari dalam program akan ditampilkan track lagu yang ada dalam CD. Pilih track yang akan dikonversi ke wav, dan lakukan proses "grab". Sebaiknya beberapa bagian dalam menu setting diganti dulu sesuai keadaan komputer. Misalnya direktori tempat file wav akan disimpan. Proses konversi sebaiknya tidak langsung menjadi file mp3, karena kita akan menggunakan proses sendiri untuk konversi ke file mp3 menggunakan lame. Juga deteksi dulu cdrom drive dengan autodetect pada bagian setting.

Setelah proses "grab" selesai, maka file wav yang sudah ada dalam harddisk akan kita "encode" menjadi file mp3. Umumnya ukuran file wav yang dihasilkan proses "grab" berkisar antara 40 MB sampai 80 MB. Namun bisa juga berukuran lebih kecil atau lebih besar. File mp3 yang akan dihasilkan berukuran sekitar 5 MB. Program lame adalah program yang berupa command line program, jadi tanpa window. Menjalankan program ini dari msdos prompt.

Untuk memudahkan, copy file lame.exe dan lame_enc.dll ke direktori tempat file wav yang akan di"encode". Ukuran kedua file ini cukup kecil, kurang dari 200 KB. Untuk melakukan proses merubah file wav ke mp3, jalankan perintah 

lame -h namafilewav.wav namafilemp3.mp3

Perintah di atas cukup menghasilkan file mp3 yang berkualitas bagus. Opsi yang tersedia pada program lame cukup banyak. Untuk detailnya bisa dibaca pada manual program tersebut.

Jika cukup banyak file yang akan di"encode" maka bisa juga dibuat sebuah file batch yang berisi perintah untuk melakukan proses "encode" semua file wav menjadi mp3. Setelah semua selesai diproses, maka kita siap untuk mendengarkan lagu-lagu kesukaan dengan player mp3. Untuk "membersihkan" ruang harddisk, file-file wav yang sudah di"encode" dapat dihapus.
