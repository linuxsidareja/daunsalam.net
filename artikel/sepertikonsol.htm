Seperti Konsol

Anda mungkin pernah melihat aplikasi jadul yang tampilannya hanya berupa teks. Ada dengan latar belakang hitam dengan teks hijau, atau latar belakang biru dengan warna teks kuning. Untuk menggunakan program tersebut diketikkan sejumlah perintah yang dikenali program. Lalu ada respon dari program yang juga dalam bentuk teks sesuai fungsi program tersebut.

Sekarang juga masih ada sejumlah aplikasi yang menggunakan pendekatan dengan cara seperti di atas. Sedikit iseng, saya membuat aplikasi berbasis web yang menirukan model di atas. Dengan menggunakan sedikit ramuan AJAX tanpa XmlHttpRequest, bentuk ini adalah penggabungan nuansa nostalgia dan AJAX yang sedang banyak didengung-dengungkan :)

Jurusnya tidaklah terlalu rumit, kalau tidak bisa dibilang sederhana. Bentuk ini disusun pada satu halaman HTML. Sebuah input berupa textbox digunakan untuk menulis perintah yang akan diproses. Sebuah textarea yang akan menjadi "layar" untuk menampilkan respon dari pemberian perintah. Sebuah frame yang berfungsi sebagai saluran untuk mengirim perintah dan menerima respon. Frame ini disembunyikan dengan CSS sehingga tak terlihat pada browser. Untuk tampilan agar terlihat seperti aplikasi dalam mode konsol, digunakan CSS untuk mengaturnya.

Cara kerjanya kira-kira seperti berikut :

  • Perintah diketik pada input textbox.
  • Form yang berisi perintah ini dikirim dengan target adalah frame yang disembunyikan.
  • Sebuah script PHP menerima kiriman form ini.
  • Perintah yang diterima diproses oleh PHP, lalu dibuat sebuah halaman HTML. Halaman HTML ini berisi javascript yang berguna untuk menampilkan hasil proses ke textarea (yang berfungsi sebagai layar respon) yang ada di halaman utama.

Demikian cara kerjanya, dilakukan berulang-ulang, sesuai perintah yang dikirim oleh pengguna. Untuk mencoba silahkan lihat halaman demo berikut ini. Dan kalau tertarik, source code yang berukuran sekitar 2 KB (zip) bisa juga didownload.
