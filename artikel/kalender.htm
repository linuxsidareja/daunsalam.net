Fungsi Kalender

Fungsi kalender ini berguna untuk menampilkan sebuah kalender pada bulan tertentu. Dibuat dalam bahasa PHP (untuk kalender dengan javascript lihat di sini). Jika anda membutuhkan sebuah kalender sederhana pada halaman situs, maka fungsi ini bisa jadi cukup berguna untuk keperluan tersebut.

kalender.txt : File yang berisi source code fungsi kalender. Download file tersebut, dan simpan dengan nama fkalender.php

Berikut ini adalah contoh penggunaan fungsi tersebut :

<html>
<head>
<style>
.tr_judul {
  font-weight : bold;
  text-align : center;
  background : #d0d0d0;
}
.tr_terang {
  text-align : center;
  background : #f0f0f0;
}
.tabel_data {
  background : #d0d0d0;
}
</style>
</head>
<body>

<?php
include "fkalender.php";
echo buatkalender(12,3,2005); /* menampilkan kalender */
?>

</body>
</html>

Dari contoh di atas Untuk menampilkan kalender, digunakan :

echo buatkalender(12,3,2005);

Argumen pertama dari fungsi buatkalender (pada contoh di atas 12) merupakan tanggal. Argumen kedua adalah bulan, yaitu 3, dan argumen terakhir adalah tahun. Sehingga pemanggilan fungsi buatkalender(12,3,2005) akan menghasilkan kalender bulan Maret 2005, dengan menampilkan tanggal 12 secara bold. Hasilnya seperti berikut ini.

Untuk mengganti tampilan kalender, dapat dilakukan melalui perubahan pada style. Kalender ini selanjutnya bisa dikembangkan dengan menambahkan fungsi navigasi, serta menampilkan nama bulan dan tahun dari kalender tersebut. Fungsi kalender ini memang belum menampilkan nama bulan dan tahun dari kalender yang ditampilkan.
