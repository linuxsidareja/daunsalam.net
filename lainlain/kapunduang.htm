Kapunduang

Kapunduang adalah sebuah program blog dibuat dalam bahasa PHP. Penyimpanan data dalam file teks sederhana. Ide program ini adalah sederhana, minimalis, dan primitif. Nah, tentang ide itu saya juga tak paham-paham betul maksudnya, jadi terima saja. Tujuan utamanya adalah untuk bahan latihan bagi yang belajar PHP, karena kode program dibuat cukup sederhana, dilengkapi sejumlah komentar yang bisa membantu untuk memahami kode. Beberapa keterangan tentang program ini :

  • Menampilkan artikel di halaman depan, dapat ditentukan jumlahnya
  • Komentar dari pengunjung, tanpa verifikasi, sehingga memiliki kelemahan bisa di-spam
  • Administrasi artikel, berupa upload, pemindahan data, rename, delete, view
  • Single user
  • Tidak ada proteksi halaman administrasi, jika diperlukan bisa memanfaatkan proteksi direktori dari webserver
  • Lisensi GNU GPL, walaupun saya belum pernah membaca lisensi ini secara keseluruhan :)
  • Berjalan dengan PHP > 4.3.xx
