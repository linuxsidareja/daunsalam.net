  • Tentang JSChord
    Program ini dibuat karena saya tak berhasil menemukan sebuah program sederhana untuk menampilkan chord piano/keyboard. Ada sejumlah diagram lengkap, tapi karena terlalu besar ukurannya, cukup repot untuk dilihat pada layar komputer. Ada juga beberapa program, tapi tampaknya terlalu ribet. Akhirnya saya putuskan untuk membuat sebuah program yang bisa menampilkan chord piano/keyboard yang hanya memerlukan satu buah file html yang dilihat dengan webbrowser. Jika anda menemukan kesalahan, ingin membantu mengembangkan, atau sekedar ucapan terima kasih, silahkan kirim email ke yan@daunsalam.net
